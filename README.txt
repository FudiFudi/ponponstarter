# 🎵 PonPonStarter

ダンス用音源に「3カウント（ぽんぽんぽん）」を自動で付与するブラウザツールです。
サビ始まり・頭出ししづらい楽曲でも、気持ちよくスタートできます。

---

## 🌐 デモ

https://あなたのユーザー名.github.io/リポジトリ名/

---

## ✨ 機能

- 音源アップロード
- 3カウント自動付与
- BPM指定（未入力時は120）
- 微調整スライダー（±0.75秒）
- 再生 / 停止
- WAVダウンロード

## 📁 必要ファイル

index.html
drum_3counts.wav
※ drum_3counts.wav は120BPMの3カウント音源

---

## 🚀 使い方

1. 音源を選択
2. BPMを入力
3. 「生成＆再生」
4. 微調整で気持ちよく合わせる
5. ダウンロード

---

## ⚠️ 注意

- 50MB以上のファイルはいれないこと
- ビート検出はしていません（手動微調整前提）
