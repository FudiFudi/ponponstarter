# 🎵 PonPonStarter

ダンス用音源に「3カウント（ぽんぽんぽん）」を自動で付与するブラウザツールです。
サビ始まり・頭出ししづらい楽曲でも、気持ちよくスタートできます。

---

## 🌐 デモ

https://fudifudi.github.io/ponponstarter/

---

## 🌐 動作確認済み環境

- windows11 + Chrome

---

## ✨ 機能

- 音源アップロード
- 3カウント自動付与
- BPM指定（未入力時は120）
- 微調整スライダー（±0.75秒）
- 再生 / 停止
- WAVダウンロード

## 📁 必要ファイル

index.html
drum_3counts.wav
※ drum_3counts.wav は120BPMの3カウント音源

---

## 🚀 使い方

1. 音源を選択
2. BPMを入力
3. 生成＆再生でテンポ確認
4. 微調整で気持ちよく合わせる
5. ダウンロード

---

## 🔒 セキュリティ

- 音源はサーバーに送信されません
- すべてブラウザ内で処理されます

---

## ⚠️ 注意

- 50MB以上の大容量ファイルを入力しないこと
- ビート検出はしていません（手動微調整が前提）
